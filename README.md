# Checkbox Automata

## What is it

A simple sandbox app ([lives here](https://astynax.github.io/checkbox-automata)) where you can click some cells of the pixel grid and watch interesting patterns, symmetrical and not. Patterns like this one:

![screenshot](./assets/example.png)

## The name

1. my first quick-and-dirty prototype was using the grid of checkboxes
2. firstly I'd planned to add some actual Cellular Automata simulation but later I decided to keep it interactive-only
